■■■■最初に

	Dotmatrix Soft をご愛顧賜り誠にありがとうございます。

	この ReadMe ファイルでは、本ソフトに関する基本的な情報が掲載さ
	れています。

	本ソフトは、UNIX・Linuxサーバー上において使用できます。

■■■■ソフト説明

このソフトは、住所録データーベースソフトです。
CSVファイルに書き込まれたアドレスデータを検索表示するものです。
データの追加はエクセルで簡単に行えるメンテナンス性を考慮したものです。

----------------------------------------------------------------------

■■■■推奨動作環境

	UNIX・Linux OSにおけるCGI動作可能サーバー。
	ご使用のプロバイダー、レンタルサーバーにおいてご確認ください。


----------------------------------------------------------------------

■■■■必要ソフト

	Perl

----------------------------------------------------------------------

■■■■インストール・アンインストールの方法

☆インストール

　　◆圧縮ファイルから解凍

	１）解凍後、address.csvファイルを開いてデータ入力を行う。
	２）FTPでアップロード
	３）kensaku.cgiのパーミッションを755に設定
	４）kensaku.htmファイルにブラウザからアクセス
	

☆アンインストール

	アップロードしたフォルダを、中身ごと全て削除します。
	これでアンインストール完了です。
